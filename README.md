# ATLien: Welcome to Atlanta

## Introduction

This App is a small introductory of the many things you can do in Atlanta. This is a one stop shop planning for all the places you would want to visit when you come to Atlanta.

## Technologies Used
- React
- MongoDB
- Express
- Node.js
- JavaScript
- HTML
- CSS
- Heroku
- Netlify
